INTRODUÇÃO

#SEMÂNTICA

Na programação, a Semântica refere-se ao significado de um pedaço de código - por exemplo "que efeito tem a execução daquela linha de JavaScript?" ou "que propósito ou função esse elemento HTML tem" (em vez de "como é ?".) ​

Atualmente o papel da HTML não é apenas estruturar documentos para a web, mas também descrever o significado do conteúdo presente nesses documentos por meio de tags semânticas. O HTML semântico tem como objetivo descrever o significado do conteúdo presente em documentos HTML, tornando-o mais claro tanto para programadores quanto para browsers e outras engines que processam essa informação.

HTML (HyperText Markup Language) é um linguagem de marcação muito conhecida e utilizada na criação de páginas web. Ela renderiza informações nos navegadores de internet juntamente com o CSS, JavaScript e outras linguagens de desenvolvimento web, permitindo que nós tenhamos acesso a todo o conteúdo de um site.

O HTML é como o esqueleto que dará forma às páginas. É a estrutura mais básica. Ele setoriza as diversas partes da página de acordo com o seu conteúdo, ou seja, ajuda a endereçar cada um dos elementos que a constitui, como os textos, imagens, menu de navegação, links, dentre outros. Posteriormente ele irá auxiliar a aplicação de outras funcionalidades a um site.

Tags HTML e seus Significados

  1. <!DOCTYPE>: define o tipo de documento
  2. <html>: define o início do documento HTML, todas as demais tags devem estar dentro desta tag
  3. <head>: cabeçalho onde ficam os metadados do documento, como meta tags, CSS e JavaScript
  4. <title>: define o título do documento, mostrado na barra de títulos do navegador e como link nos mecanismos de busca
  5. <base>: define a URL base do documento
  6. <link>: utilizado para linkar um arquivo de imagem ou CSS à página HTML
  7. <script>: utilizado para escrever ou anexar um arquivo normalmente JavaScript à página
  8. <meta>: meta tags, como charset, description, viewport ou keywords
  9. <style>: utilizada para adicionar um código CSS à página
  10. <body>: corpo ou área principal do documento, dentro deste deve ficar o conteúdo da sua página
  11. <header>: define um cabeçalho da página ou de uma seção, normalmente pode conter um menu (nav), um logo ou link
  12. <nav>: define uma área de navegação contendo links, para a formação de menus com hiperlinks
  13. <footer>: define o rodapé de uma página ou seção
  14. <main>: define o conteúdo principal da página, deve ser utilizado apenas uma única vez em cada página
  15. <section>: define uma seção do documento
  16. <article>: pode ser um post ou notícia de um fórum ou blog, define um conteúdo independente
  17. <h1> a <h6>: representam 6 níveis de títulos, sendo o <h1> de maior importância
  18. <p>: define um parágrafo
  19. <br>: define uma quebra de linha
  20. <div>: define uma camada sem formatação, genérico
  21. <ul>: define uma lista não ordenada
  22. <li>: define um elemento ou item da lista não ordenada
  23. <a>: define um link que deve ser apontado com o atributo href
  24. <img>: define a utilização de uma imagem (JPG, GIF, PNG e outras). A imagem deve ser apontada com o uso do atributo src
  25. <strong>: apesar de deixar em negrito, representa a importância deste texto no meio da frase
  26. <span>: utilizado para definir pequenos trechos de textos
  27. <pre>:exibe o texto exatamente da forma que foi digitado, incluindo tabulações, espaços e enters

Conhecendo os elementos semânticos do HTML5

<meta>

A <meta> ajuda o seu site a descrever seu conteúdo aos buscadores, como o Google. Aqui você poderá dizer qual a codificação (charset), descrição e até palavras-chaves referentes a sua página. Essas são importantes justamente por identificar o site aos buscadores, ajudando no SEO (Search Engine Optimization). O Google não utiliza somente as metas para realizar suas buscas, mas elas são uma parte importante.

  • description: guarda a descrição do site, utilizado nos motores de busca.
  • keywords: palavras chaves relacionadas ao site, utilizada nos primórdios pelos motores de busca.
  • author: identifica o autor do site
  • robots: ajuda os robôs dos motores de busca a indexar ou não um site. Com esta opção podemos dizer se uma determinada página do site não deve ser mostrada nos resultados de uma busca.
Exemplo de código: 
    
      <head>
      <title>Página do Burnes - Home </title>
      <meta charset="utf-8">
      <meta name="description" content="Página do Burnes sobre HTML e CSS">
      <meta name="keywords" content="html, css, js, site">
      <meta name="robots" content="index">
      </head>

<nav>

O elemento <nav> representa um menu de navegação na sua página. Normalmente é o primeiro elemento da nossa página, mas independentemente da posição que você escolher para a sua navbar é boa prática envolver o agrupamento de links com esta tag.

Exemplo de código: 
    
      <nav>
      <ul>
       <li> <a> href="”#”">pagina 1</a> </ul>
       <li> <a> href="”#”">pagina 2</a> </ul>
       <li> <a> href="”#”">pagina 3</a> </ul>
       <li> <a>href="”#”">pagina 4 </a> </ul>
      </ul>
      </nav>

<section>

O elemento <section> representa uma seção dentro de um documento e geralmente contém um título, o qual é definido por meio de um dos elementos entre <h1> e <h6>. Podemos utilizar o <section> , por exemplo, para descrever as seções/tópicos de um documento.

Exemplo de código: 
    
      <section>
       <h3> Seção 1 </h3>
       <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do 
       eiusmod ...
       </p>
      </section>

<iframe>

O elemento HTML <iframe> representa um contexto de navegação aninhado, incorporando outra página HTML na atual. iFrame é a abreviação de Inline Frame. É um elemento poderoso quando se trata de web design. Você provavelmente já viu diversos vídeos do YouTube que estavam incorporados em outros sites. Para além disso, um iFrame pode inserir todo tipo de mídia.

Atributos:
  • O iFrame source (src) é a origem do conteúdo, seja do servidor interno o externo. Não se esqueça de incorporar o código na url.
  • Width (largura) e height (altura) são os aspectos visuais. Você pode inserir uma medida fixa como 680×480 pixels (px) no exemplo. Ou, você pode usar o método de porcentagem (10%-100%) na qual o conteúdo será ajustado automaticamente.
Exemplo de código: 
    
      <iframe>
       id="inlineFrameExample"
       title="Inline Frame Example"
       width="300"
       height="200"
       src="url.link">
      </iframe>
